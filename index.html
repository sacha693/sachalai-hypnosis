關於我與催眠的奇妙之旅

對於生命意義的探索

  • 因孩子與生命教育結緣

    我因孩子成為學校志工，開始接觸生命教育，成為一位說故事的人，在這之中我開始重新探索自己「想成為什麼樣的人？」以及「想過什麼樣的生活？」

  • 與「催眠」的偶然相遇

    一次偶然的機會，我遇見了催眠，深深愛上這種能調整潛意識的魔力，理解到力量就在自己身上而不是期盼別人給予，從而下定決心，也要成為能陪伴他人的催眠療癒師。

  • 持續學習與自我療癒

    拿到NGH催眠師證照後，依然持續學習並拿取催眠相關證照（IACT艾瑞克森現代催眠、ABNLP美國神經語言程式、希塔療癒），因為在學習療癒別人的同時，也溫柔地療癒著自己。

  • 我的核心理念：潛意識調整

    我喜歡稱這種繞過表意識所做的工作為 **潛意識調整**，因為我們所有的外在行為，都源自內在深處的信念。而催眠，正是幫助我們連結、調整與改寫這些信念的途徑。

我能為你做什麼？

           

找回平靜的旅程 +

生活中的壓力常常讓我們喘不過氣。我會溫柔地引導你，在一個完全放鬆的狀態下，釋放身心的緊繃與疲憊，重新找回內心的平靜與和諧。

潛意識的習慣重塑 +

不論是想培養一個新習慣，還是告別不再適合你的舊習慣，我們都將在潛意識的深處，植入正向且支持性的新信念，讓改變變得更輕鬆、更自然。

點亮內在的自信之光 +

當你對自己感到懷疑時，內在的光芒就會被遮蔽。我會引導你釋放內在的恐懼，重新連結你的力量。從此，不論面對任何挑戰，都能展現出你最真實、最有自信的樣子。

與情緒溫柔和解 +

有時，情緒像洶湧的浪潮。我將陪伴你安全地探索情緒的源頭，溫柔地處理那些被壓抑的情感。讓你看見情緒背後的訊息，從而獲得真正的平靜與和解。

回溯過往的記憶之河 +

我們將安全地回到過往的記憶，重新理解那些曾經影響你的事件與情緒。這趟旅程將幫助你釋放被卡住的能量，讓過去不再阻礙你走向更光明的未來。

潛意識的內在住所探索 +

你的心靈如同一個美麗的花園。我將引導你進入你的潛意識內在住所，一同探索每一個角落。藉由調整住所裡的象徵物，我們能對應調整現實生活中的狀態，讓你的內外世界都更加美好。
